Privacy Policy

How we collect, use, and protect your data

Data We Collect

  • Full name, email, phone number, date of birth, gender, address
  • Professional information: qualifications, experience, skills
  • Resume and cover letter content
  • Payment transaction ID (NOT card details — handled by Razorpay)
  • Session cookies for authentication

How We Use Your Data

  • Processing job applications only
  • Sending notifications on application status
  • Processing ₹50 fee and refunds
  • We NEVER sell data to third parties

Payment Data Security

  • All payments processed via Razorpay (PCI-DSS Level 1 certified)
  • We do not store any card or banking details on our servers
  • Only Razorpay Payment ID and Order ID are stored

Data Retention Policy

  • Hired candidates: data retained for 2 years
  • Rejected/Not Selected candidates: data deleted after 90 days
  • Resume files: encrypted at rest on AWS S3
  • Candidates can request data deletion at privacy@damorhr.com

Candidate Rights

  • Right to access personal data collected
  • Right to edit or update profile information
  • Right to request complete data deletion
  • Right to know refund status at any time

Refund Policy

Our transparent refund process

₹50 Fully Refundable
  • ₹50 application fee is fully refundable if candidate is not selected
  • Refund is triggered automatically when HR marks status as "Rejected"
  • Refunds are processed within 5–7 business days to the original payment source
  • Candidates receive email + SMS confirmation when refund is initiated
  • If hired, the ₹50 fee is non-refundable and treated as a processing charge
  • For disputes, contact refunds@damorhr.com within 30 days
