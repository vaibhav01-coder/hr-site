1. Project Overview
The HR Hiring Portal is a full-stack web application that enables companies to post jobs and allows candidates to apply online. The platform includes a refundable ₹50 application fee system — candidates pay before submitting, and if not selected, the fee is automatically refunded within 5–7 business days.

1.1 Core Purpose
Allow HR/Admin to post and manage job listings
Allow candidates to register, browse jobs, and apply
Collect ₹50 refundable fee before application submission
Auto-refund rejected candidates via Razorpay API
Maintain full privacy compliance (GDPR + Indian IT Act 2000)

2.3 Key Components
2.3.1 ApplicationForm Component
A multi-step form collecting:
Step 1 — Personal Info: Name, Email, Phone, DOB, Gender, Address
Step 2 — Professional Info: Qualification, Experience, Current Title, Skills, LinkedIn
Step 3 — Resume Upload (PDF/DOC, max 5MB) + Cover Letter
Step 4 — Preview & Submit (disabled until payment confirmed)

2.3.2 PaymentGateway Component
Handles ₹50 Razorpay checkout flow:
Loads Razorpay script dynamically
Creates order via backend API call
Opens Razorpay modal with pre-filled candidate details
On success: stores paymentId, enables application submission
Displays security trust badges (PCI-DSS, SSL icons)

2.3.3 StatusTracker Component
Stepper UI shown on candidate dashboard:
Step 1 — Application Submitted
Step 2 — Under Review
Step 3 — Shortlisted / Rejected
Step 4 — Hired / Refund Initiated
Step 5 — Refund Processed (3–5 business days)

3.4 Payment & Refund Flow
Step-by-step Razorpay Integration:
1. Frontend calls POST /api/payment/create-order
2. Backend creates Razorpay order (amount: 5000 paise = ₹50)
3. Frontend opens Razorpay checkout modal
4. On success, frontend calls POST /api/payment/verify
5. Backend verifies HMAC signature using Razorpay secret
6. On verification success: application is unlocked for submission
7. When HR marks as Rejected: backend calls Razorpay Refund API
8. Refund confirmation sent via email + SMS to candidate
9. Refund timeline: 3–7 business days to source account

4. Privacy Policy & Legal Pages
4.1 Privacy Policy  — /privacy-policy
Data We Collect
Full name, email, phone number, date of birth, gender, address
Professional information: qualifications, experience, skills
Resume and cover letter content
Payment transaction ID (NOT card details — handled by Razorpay)
Session cookies for authentication

How We Use Your Data
Processing job applications only
Sending notifications on application status
Processing ₹50 fee and refunds
We NEVER sell data to third parties

Payment Data Security
All payments processed via Razorpay (PCI-DSS Level 1 certified)
We do not store any card or banking details on our servers
Only Razorpay Payment ID and Order ID are stored

Data Retention Policy
Hired candidates: data retained for 2 years
Rejected/Not Selected candidates: data deleted after 90 days
Resume files: encrypted at rest on AWS S3
Candidates can request data deletion at privacy@[yoursite].com

Candidate Rights
Right to access personal data collected
Right to edit or update profile information
Right to request complete data deletion
Right to know refund status at any time

4.2 Refund Policy  — /refund-policy
₹50 application fee is fully refundable if candidate is not selected
Refund is triggered automatically when HR marks status as 'Rejected'
Refunds are processed within 5–7 business days to the original payment source
Candidates receive email + SMS confirmation when refund is initiated
If hired, the ₹50 fee is non-refundable and treated as a processing charge
For disputes, contact refunds@[yoursite].com within 30 days
